Tournaments in Bolt, are a way for people to test out their skills against other players in a competitive environment,
Tournaments are a great way to earn rewards and show off their skills.

## Commands

| Command | Description                       |
|---------|---------------------------------- |
| /tournament help                            | Bring up help menu for Tournaments 
| `/tournament host <type>`                   | Host a tournament by a player 
| `/tournament chost <type> <kit> <teamsize>` | Host a tournament through console 
| /tournament forceStart                      | Force start the current tournament 
| /tournament info                            | View the information for the current tournament 
| /tournament join                            | Join the tournament 
| /tournament leave                           | Leave the tournament 
| /tournament cancel                          | Cancel the current tournament
