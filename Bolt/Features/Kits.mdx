## Commands

`<>` = Required `[]` = Optional

Command                             | Permission | Description
----------------------------------- | ---------- | -------------------------------------
`/kit addEndCMD <kit> <text>`       | ``         | Adds a match end command.
`/kit addStartCMD <kit> <text>`     | ``         | Adds a match start command.
`/kit create <name>`                | ``         | Creates a kit.
`/kit delete <kit>`                 | ``         | Deletes a kit.
`/kit deleteAll`                    | ``         | Deletes all kits.
`/kit editor`                       | ``         | Opens the kit editor.
`/kit getEditInv <kit>`             | ``         | Loads a kit's edit inventory.
`/kit getInv <kit>`                 | ``         | Loads a kit's inventory.
`/kit help [page]`                  | ``         | Displays the help menu.
`/kit info <kit>`                   | ``         | Displays information about a kit.
`/kit list [page]`                  | ``         | Lists all kits.
`/kit manage`                       | ``         | Opens the kit manage menu.
`/kit save`                         | ``         | Saves all kits.
`/kit setBotKb <kit> <profile>`     | ``         | Sets a kit's bot knockback profile.
`/kit setEditInv <kit>`             | ``         | Sets a kit's edit inventory.
`/kit setIcon <kit>`                | ``         | Sets a kit's icon.
`/kit setInv <kit>`                 | ``         | Sets a kit's inventory.
`/kit setKb <kit> <profile>`        | ``         | Sets a kit's knockback profile.
`/kit setName <kit> <name>`         | ``         | Sets a kit's display name.
`/kit setPriority <kit> <priority>` | ``         | Sets a kit's priority.
`/kit wipeKits kit <kit>`           | ``         | Wipes everyones kit editor for a kit.
`/kit wipekits player <player>`     | ``         | Wipes a players kit editor.
