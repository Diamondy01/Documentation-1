The notes system allows staff members to add specific text to a players profile. 
For example you can mark siblings so staff members don't IP Ban them.

## Commands

`<>` = Required `[]` = Optional

Command                       | Permission         | Description
----------------------------- | ------------------ | ----------------------------
`/notes <player> [page]`      | core.command.notes | View a players notes.
`/notes add <player> <note>`  | core.command.notes | Add a note to a player.
`/notes remove <player> <id>` | core.command.notes | Remove a note from a player.
