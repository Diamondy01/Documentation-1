Disguises allow you to disguise/nick yourself to avoid detection by other people

This feature if useful for content creators or celebrities on large servers

## Player Commands

`<>` = Required `[]` = Optional

Command                   | Permission | Description
------------------------- | ---------- | ---------------------------------------
`/disguise(/nick) `       | None       | Allows you to set your nick/disguise.
`/undisguise(/unnick) `   | None       | Allows you to unnick/undisguise.



## Admin Commands

`<>` = Required `[]` = Optional

Command                                                  | Permission                              | Description
-------------------------------------------------------- | --------------------------------------- | ---------------------------------------
`/disguiseadmin checkdisguise <player>`                  | core.command.disguiseadmin.checkdisguise| Allows you to check a players disguise.
`/disguiseadmin disguise <rank> <player> <skin> <target>`| core.command.disguiseadmin              | Allows you to set a disguise.
`/disguiseadmin help [page]`                             | core.command.disguiseadmin              | Brings up the disguiseadmin help menu.
`/disguiseadmin history <player>`                        | core.command.disguiseadmin.history      | Allows you to check the history of a player.
`/disguiseadmin listdisguised`                           | core.command.disguiseadmin.listdisguised| Allows you to list disguised players.
`/disguiseadmin undisguise <player>`                     | core.command.disguiseadmin.undisguise   | Allows you to undisguise a specified player.
