The chat logs system provides an easy way to track players' chat history. 
You can check the chat logs of a specific player or with a specific message.

## Commands

`<>` = Required `[]` = Optional

Command                               | Permission                    | Description
------------------------------------- | ----------------------------- | --------------------------------------------
`/chatlogs view <player>`             | `core.command.chatlogs`       | View a player's chat logs.
`/chatlogs search <player> <message>` | `core.command.chatlogs`       | View a player's chat logs with that message.
`/chatlogs clear <player>`            | `core.command.chatlogs.clear` | Clear a player's chat logs.
