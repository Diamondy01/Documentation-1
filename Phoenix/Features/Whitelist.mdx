The Whitelist system allows you to protect your server against unwanted attention.

It allows you to make your server more private and keep it secure.

## Commands

`<>` = Required `[]` = Optional

Command                                           | Permission               | Description
------------------------------------------------- | ------------------------ | --------------------------------
`/whitelist access <access [Rank/UUID]> [rank]`   | `core.command.whitelist` | Allows you to choose if ranks(Ranks) or players(UUID) have access.
`/whitelist add <player> <permanent [true/false]>`| `core.command.whitelist` | Allows you to add a player to whitelist.
`/whitelist help`                                 | `core.command.whitelist` | Brings up the help menu.
`/whitelist info`                                 | `core.command.whitelist` | Brings up the info page.
`/whitelist remove <player>`                      | `core.command.whitelist` | Removes a player from whitelist.
`/whitelist toggle [true/false]`                  | `core.command.whitelist` | Allows you to toggle whitelist.
