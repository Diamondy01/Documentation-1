## Creating kits

1. Use `/kit create <name>` to create a new kit
2. Use `/kit setInventory <kit>` to set the inventory of the kit with items from your inventory
3. Use `/kit setArmor <kit>`  to set the armor of the kit with armor from your inventory
4. Use `/kit setDamage <kit> <true/false>` to allow losing health with this kit (Optional)
5. Use `/kit setBreaking <true/false> <allowDrops>` to allow breaking blocks with this kit (Optional)
    - allowDrops: Whether broken blocks should drop items or not
6. Use `/kit enable <kit>` to enable the kit
7. Use `/kit saveAll` to manually save all kits (Optional)
